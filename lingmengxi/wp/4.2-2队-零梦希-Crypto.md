## 2.VISA_or_MC-HackINI

描　　述: 

\> I've stolen a lot of credit cards from my friends,
\> but I don't know which one is a VISA and which one
\> is a MasterCard. Can you help me?
**Author**: m0kr4n3

file.txt

查资料发现里面都是卡号，VISA的卡号是4，Mastercard的是5

只有两个数字，且卡号很多，那么先把4的卡转为0，5为1

参考代码

```
with open("file.txt", "r") as f:
    result = []
    for line in f:
        card = line.strip().replace("-", "")
        if card and card[0] == "4":
            result.append("0")
        else:
            result.append("1")

print("".join(result))  
print(" ".join(result))
```
